# antenna-matcher

アンテナ給電マッチング選定ガイド + 計算ツール

**URL:** https://jq3jqo-station.github.io/antenna-matcher/

## 概要

アンテナの種類と給電点を選ぶと、最適なマッチングトランス（UnUn/Balun）を自動選定し、
コアの巻数・線材長・接続図をまとめて提示するガイドツールです。

## 対応アンテナ

| アンテナ | 給電点 | 推定インピーダンス | 変換比 | 種別 |
|---|---|---|---|---|
| ロングワイヤー | 端部 | ~500Ω | 9:1 | UnUn |
| ロングワイヤー | L型 | ~200Ω | 4:1 | UnUn |
| デルタループ | 頂点 | ~100Ω | 2:1 | Balun |
| デルタループ | 辺中点 | ~75Ω | 直結 | 1:1 Balun |
| デルタループ | 底辺 | ~200Ω | 4:1 | Balun |
| ダイポール | 中点 | ~73Ω | 直結 | 1:1 電流Balun |
